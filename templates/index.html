About

Tech companies use facial recognition technology (FRT) to learn more about their customers. They use this information to entice, manipulate, and deceive us. Modern FRT is based off deep learning, which is a type of artificial neural network that has recently revolutionized AI. A deep learning model can predict some category given an input. In this case, a model can predict a categroy like gender given a video of your face. This app uses multiple models to predict age, ethnicity, gender, and emotion. Google, Amazon, and Facebook don't need your consent or your profile information to learn all about you: they just need a picture of your face.

What is AI?

Artificial Intelligence is any system that mimics human behavior. This can be as simple as knowing that 1 plus 1 equals 2 or as complex as driving a car. Like any technology, AI has the capacity to be used for good or for evil. If you want to be able to tell the difference between the two, you should learn more about AI.

How does this project work?

The code for this project is on GitHub. All the data that this website collects is immediately thrown away: there's no database or server collecting your information: it's simply streamed back to your browser and is gone forever. This was created by Cameron Smith.
